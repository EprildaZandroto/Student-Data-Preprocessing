# Student Data Preprocessing

## Deskripsi
Project ini merupakan implementasi preprocessing data menggunakan Python dan Scikit-learn.

## Tahapan
- Import library
- Membaca dataset
- Encoding data kategorikal
- Scaling menggunakan MinMaxScaler
- Feature Engineering
- Analisis Korelasi
- Visualisasi Heatmap

## Tools
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Google Colab

## Dataset
Dataset berisi data mahasiswa dengan atribut:
- Usia
- Jam Belajar
- Kehadiran
- Tugas
- Nilai Ujian
- Ekstrakurikuler
- Status Ekonomi
- Lulus
